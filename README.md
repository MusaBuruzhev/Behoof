# Behoof - Платформа сравнения цен на электронику

Behoof - современная веб-платформа для сравнения цен и характеристик электроники. Приложение предоставляет удобный интерфейс для управления каталогом товаров с поддержкой истории цен, детальных характеристик и интерактивных графиков.

## 🚀 Возможности

- **Управление товарами**: Добавление, обновление и удаление товаров через удобный интерфейс
- **История цен**: Автоматическое отслеживание изменений цен с интерактивным графиком
- **Детальная страница товара**: Полная информация о товаре с галереей изображений
- **Характеристики товаров**: Структурированные характеристики по категориям (экран, процессор, память и т.д.)
- **Категории товаров**: 8 основных категорий электроники с подкатегориями
- **Графики и визуализация**: ApexCharts для отображения истории цен
- **Современный UI**: Адаптивный дизайн с использованием Vue 3 и современных технологий

## 🛠 Технологии

### Фронтенд
- **Vue 3** - Composition API
- **Vue Router** - SPA навигация
- **Vite** - Быстрая сборка
- **Axios** - HTTP клиент
- **ApexCharts + vue3-apexcharts** - Интерактивные графики и диаграммы

### Бэкенд
- **Node.js + Express** - REST API
- **MongoDB + Mongoose** - Документоориентированная БД
- **Multer** - Загрузка файлов

## 🚀 Установка и запуск

### 1. Клонирование
```sh
git clone https://github.com/MusaBuruzhev/Behoof.git
cd Behoof
```

### 2. Установка зависимостей
```sh
npm install
cd backend && npm install && cd ..
```

### 3. Настройка БД
Создайте файл `backend/.env`:
```env
PORT=5000
MONGODB_URI=mongodb://localhost:27017/behoof
```

### 4. Запуск
```sh
cd backend && npm run dev &
npm run dev
```

Приложение доступно на http://localhost:5173

## 📚 API

### Основные эндпоинты
- `GET /api/catalog` - Получить каталог
- `GET /api/products` - Получить список товаров (поддерживается пагинация и фильтры, см. ниже)
- `POST /api/products` - Добавить товар
- `PUT /api/products/:id` - Обновить товар
- `DELETE /api/products/:id` - Удалить товар
- `GET /api/products/:id` - Получить товар

### Параметры для `GET /api/products`
Можно передавать query-параметры для фильтрации и пагинации:

- `page` (number) – номер страницы (по умолчанию 1)
- `limit` (number) – количество элементов на странице (макс. 100, по умолчанию 20)
- `q` (string) – текстовый поиск по имени, бренду и описанию
- `categoryId`, `subcategoryId`, `modelId` – фильтр по иерархии каталога
- `priceMin`, `priceMax` – диапазон цены
- `brand` – бренд или список брендов через запятую
- `sortBy` – сортировка: `price-asc`, `price-desc`, `name-asc`, `name-desc`, `date-desc`

## 🎨 Функционал

### Управление товарами
- Добавление товаров с характеристиками
- Обновление цен (автоматическая история)
- Редактирование характеристик
- Удаление товаров

### Детальная страница товара
- **Галерея изображений**: Адаптивная галерея с миниатюрами и стрелками навигации
- **Полная информация**: Название, бренд, цена, описание, характеристики
- **Рейтинги характеристик**: Визуальные сегментные рейтинги вместо звезд
- **История цен**: Интерактивный график изменений цены со временем
- **Адаптивный дизайн**: Оптимизирован для всех устройств

### График истории цен
- **Библиотека**: ApexCharts + vue3-apexcharts для Vue 3
- **Тип графика**: Area chart с плавными линиями
- **Визуализация**: Градиентная заливка, маркеры точек
- **Интерактивность**: Tooltip с датой и ценой при наведении
- **Условия отображения**: Показывается только при наличии 5+ точек данных
- **Адаптивность**: Полностью responsive дизайн

### Навигация и UX
- **Vue Router**: SPA навигация без перезагрузки страницы
- **Кликабельные карточки**: Переход к деталям из любого места каталога
- **Обратная связь**: Визуальные индикаторы загрузки и ошибок
- **Современный UI**: Использование Vue 3 Composition API

### Характеристики по категориям
- **Смартфоны**: Экран, Процессор, Память, Камера, Аккумулятор
- **Ноутбуки**: Процессор, RAM, Диск, Экран, GPU
- **Планшеты**: Экран, Процессор, Память, Камера
- И другие категории...
